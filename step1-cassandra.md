<div class="step-title">Connect to Cassandra and create a keyspace</div>

✅ Start Cassandra:
```
./cassandra
```

✅ Start the CQL shell:
```
cqlsh
```

✅ Create the `sensor_data` keyspace:
```
CREATE KEYSPACE sensor_data
WITH replication = {
  'class': 'NetworkTopologyStrategy', 
  'DC-Houston': 1 };
```

✅ Set the current working keyspace:
```
USE sensor_data;
```
